参考文献与数据来源

本站的 DIAAS 值来自下列同行评议文献；所显示的氨基酸数值取自 USDA FoodData Central。「食物成分数据来源」下列出的各国家与地区食物成分数据库，是食物蛋白质与氨基酸组成的权威参考。现有整食物的 DIAAS 为整理自 Mathai 2017、Bailey 2020 与 FAO 2013 的代表值。
