参考文献与数据来源

本站所有 DIAAS 值均来自下列同行评议文献；氨基酸组成来自 USDA FoodData Central。现有整食物的 DIAAS 为整理自 Mathai 2017、Bailey 2020 与 FAO 2013 的代表值。

  1. Mathai JK, Liu Y, Stein HH. 某些乳制品与植物蛋白的可消化必需氨基酸评分（DIAAS）或比 PDCAAS 更能描述蛋白质量。Br J Nutr. 2017;117(4):490-499。
    https://pubmed.ncbi.nlm.nih.gov/28382889/
  2. Bailey HM, Mathai JK, Berg EP, Stein HH. 多数肉制品的 DIAAS 大于 100，但加工可能提高或降低蛋白质量。Br J Nutr. 2020;124(1):14-22。
    https://pubmed.ncbi.nlm.nih.gov/32089140/
  3. Herreman L, Nommensen P, Pennings B, Laus MC. 基于可消化必需氨基酸评分对植物源与动物源蛋白质量的全面综述。Food Sci Nutr. 2020;8(10):5379-5391。
    https://doi.org/10.1002/fsn3.1809
  4. Nosworthy MG, 等。熟制加拿大豆类的蛋白质量测定。Food Sci Nutr. 2017;5(4):896-903。
    https://doi.org/10.1002/fsn3.473
  5. Han F, 等。九种熟制谷物的可消化必需氨基酸评分。Br J Nutr. 2019;121(1):30-41。
    https://doi.org/10.1017/S0007114518003033
  6. Hammer L, 等。黄粉虫幼虫与蟋蟀体外总蛋白消化率高，按体外 DIAAS 评定可达良至优蛋白质量。Front Nutr. 2023;10:1150581。
    https://doi.org/10.3389/fnut.2023.1150581
  7. 联合国粮农组织（FAO）。《人类营养中的膳食蛋白质质量评价》。FAO 食品与营养文集第 92 号。罗马：FAO；2013。
    https://www.fao.org/ag/humannutrition/35978-02317b979a686a57aa4593304ffc17f06.pdf
  8. 美国农业部农业研究局。FoodData Central 食物成分数据库。
    https://fdc.nal.usda.gov
